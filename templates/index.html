Authentication Endpoints

POST Login

Authenticate user with email and password

/login

Request

{
  "email": "user@example.com",
  "password": "securepassword123"
}

Response (Success)

{
  "status": "success",
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "email": "user@example.com"
}

POST Signup

Create a new user account

/signup

Request

{
  "name": "Jane Smith",
  "email": "jane@example.com",
  "password": "strongpassword!123"
  "username": "janesmith"
}

Response (Success)

{
  "status": "success",
  "message": "Please verify OTP.",
  "email": "demo@email.com"
}

POST Verify OTP

Verify the OTP sent to user's email

/verify

Request

{
  "email": "user@gmail.com",
  "otp": "784532"
}

Response (Success) 

{
  "status": "success",
  "message": "OTP verified successfully"
}

POST Complete Signup

Finalize user registration after OTP verification

/complete-signup

Request

{
  "email": "user@email.com",
  "OTP":"784532"
}

Response (Success)

{
  "status": "success",
  "message": "Registration completed",
  "token": "fwewrqfr"
}

Prediction Endpoint

POST Mental Health Prediction

Predict mental health condition based on survey responses

/predict
Note: This endpoint expects an array of exactly 23 integers representing answers to the following questions in order.

Survey Questions

  1. What is your age? (Integer)
  2. What is your gender? (Male=1, Female=0, Other=15)
  3. What's your country? (Encoded as integer - see reference)
  4. Are you self-employed? (Yes=1, No=0)
  5. Do you have a family history of mental illness? (Yes=1, No=0)
  6. How often does your mental health interfere with work? (1-5 scale)
  7. How many employees are in your company? (Encoded value)
  8. Do you work remotely? (Yes=1, No=0)
  9. Do you work in a tech company? (Yes=1, No=0)
  10. Does your employer provide mental health benefits? (Yes=1, No=0, Don't know=2)
  11. Are care options available from your employer? (Yes=1, No=0)
  12. Does your employer have a wellness program? (Yes=1, No=0)
  13. Is help-seeking encouraged at your workplace? (Yes=1, No=0)
  14. Is anonymity maintained if you seek help? (Yes=1, No=0)
  15. How easy is it to take leave for mental health reasons? (1-5 scale)
  16. Are there negative consequences of disclosing a mental health issue? (Yes=1, No=0)
  17. Are there negative consequences of disclosing a physical health issue? (Yes=1, No=0)
  18. Are your coworkers supportive of mental health issues? (1-5 scale)
  19. Is your supervisor supportive of mental health issues? (1-5 scale)
  20. Would you discuss a mental health issue in an interview? (Yes=1, No=0)
  21. Would you discuss a physical health issue in an interview? (Yes=1, No=0)
  22. Do you think mental and physical health are treated equally? (Yes=1, No=0)
  23. Have you ever faced consequences for mental health issues at work? (Yes=1, No=0)

Request (Authenticated)

{
  "responses": [28, 1, 15, 0, 1, 3, 4, 1, 1, 1, 1, 0, 1, 1, 2, 0, 0, 4, 3, 0, 1, 0, 0]
}

Response (Success)

{
  "status": "success",
  "prediction": "moderate",
  "confidence": 0.78,
  "recommendations": [
    "Consider talking to a mental health professional",
    "Try mindfulness meditation exercises",
    "Schedule regular breaks during work"
  ]
}
